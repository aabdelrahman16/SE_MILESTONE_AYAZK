# Event Management Platform

## Description
A full-stack event management platform for managing events, venues, vendors, guests, bookings, invoices, and tasks.

## Technologies
- React
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Environment Variables

Create a `.env` file in `backend`:

```env
MONGO_URI=<your_mongodb_uri>
JWT_SECRET=testsecret
PORT=5001
```
